This C++ project implements an Interactive User Friendships Management System that allows users to create profiles, manage friendships, and display user information.

1. Add Users: Enter the user's details such as username, first name, last name, age, and gender. The user is added to the system if the username is not already taken.

2. Make Friends: Enter the usernames of two users to make them friends. The system establishes a bidirectional friendship between the users.

3. Display All Users: Lists all users along with their details. 

4. Display All Friendships: Lists all friendships, showing each user and their friends.

5. Exit: Ends the program.

This project provides a simple yet effective way to manage user data and relationships, making it suitable for applications that require basic social networking functionalities.
